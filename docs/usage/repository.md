# このリポジトリの使い方

---
layout: false
## github.com の操作

1. このリポジトリを Fork
1. リポジトリを好きな名前に変更
1. Settings で github.io を設定
    1. 「master docs を使う」を選ぶ
    1. テーマは好きなものを選ぶ
1. 動作確認
1. docs/ 以下を編集していく
    1. テンプレートをコピー
    1. 適宜 docs/index.html を編集(変更、追加など)
